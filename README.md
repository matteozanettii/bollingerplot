# FSDA Bollinger Analysis Tool

A robust MATLAB utility for technical financial analysis. This tool is specifically built to integrate with the **FSDA (Flexible Statistics and Data Analysis) Toolbox**, processing output from the `getYahoo` function to generate professional-grade Bollinger Band visualizations.

## Key Features
- **FSDA Integration:** Native support for the `struct` and `timetable` outputs generated by the FSDA `getYahoo` utility.
- **Technical Analysis:** 20-period Simple Moving Average (SMA) and ±2 Standard Deviation bands.
- **Visual Intelligence:** Dynamic color-coding (Green/Red) for price segments based on trend direction.
- **Data Normalization:** Automatic UTC time-zone alignment and handling of missing data points (NaN).
- **Summary Reporting:** Exports a summary table with the most recent price action and band limits.

## Dependencies
**Crucial:** This tool requires the **FSDA Toolbox** (v8.7.10.13 or later) to be installed in your MATLAB environment to fetch the necessary market data via the `getYahoo` function.
